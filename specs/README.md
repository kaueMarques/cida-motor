# Especificações do Projeto (Specs)

Este diretório contém os contratos de comportamento do sistema.

## Fluxo de Trabalho (SDD)
1. **Definição:** Adicione ou modifique um arquivo `.md` nesta pasta descrevendo o comportamento desejado.
2. **Implementação:** Desenvolva o código em `src/` (ou ajuste o motor existente) para atender à especificação.
3. **Validação:** Crie ou atualize um teste em `tests/spec_compliance/` que verifica se o comportamento definido na spec está sendo executado corretamente.

*Nunca implemente uma funcionalidade sem um arquivo correspondente nesta pasta.*
